# tc-on-aerial-robotics.github.io

Astro site for the Technical Committee on Aerial Robotics.

## Requirements

- Node.js 18
- npm or Yarn

## Run Locally

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Open:

```text
http://localhost:4321/tc-on-aerial-robotics/
```

## Production Check

Build the site:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Notes
- The configured base path is `/tc-on-aerial-robotics/`, so the local URL includes that path.
- `yarn install && yarn dev` also works if you prefer Yarn.
